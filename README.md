# Student Record Management System (SRMS)

## Description
This project is a Java-based Student Record Management System developed as a group coursework project.
It allows managing student records including adding, deleting, viewing, saving, and loading data from files.

## Features
- Add, update, delete, and search students
- Save and load student records from file
- Auto-save functionality using multithreading
- Report generation by GPA, major, and year
- Exception handling using custom exceptions
- UML-based object-oriented design

## Technologies Used
- Java
- OOP Concepts
- File I/O
- Multithreading
- GitHub for collaboration

## How to Run
1. Open the project in NetBeans or any Java IDE.
2. Make sure all classes are in the same package.
3. Run the `SRMSProject` class.
4. Follow the menu options in the console.
